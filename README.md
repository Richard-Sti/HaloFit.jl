# HaloFit.jl

This repository provides a set of utitilies aimed at fitting dark matter halo properties in a periodic box.

## Functions
- `periodic_distance`
- `center_of_mass`
- `shrinking_sphere_cm`
- `angular_momentum`
- `spherical_overdensity_mass`
- `lambda_bullock`
